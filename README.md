# ⚠️⚠️ Work in progress ⚠️⚠️

This is a tool that helps to create screenshot with deep of field effect in Counter Strike Global Offensive.

# setup

```
/src/app$ npm install
/src/static$ npm install
/www$ npm install
```

# build

```
/src/app$ npx webpack
/src/static$ npx gulp
/www$ npx electron-builder
```
